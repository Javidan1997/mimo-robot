Mimo · the AI companion robot platform

The desktop AI companion with a personality you can feel.

Mimo turns AI from a text box into a real companion: voice, expression, light, and motion in one character. The same personality scales from this live 3D demo to mobile apps, camera AR, and a physical desk robot.

A companion platform — not a one-off gadget

Most robots ship as a single rigid device. Mimo is one AI personality that spans web, mobile, camera AR, and hardware — so every new surface expands the audience and revenue, instead of restarting the product.

The robot body (ESP32-S3) stays simple — it only runs face, servo, light, and sound commands. All the heavy AI lives in a backend brain. That split lets the platform ship software fast and upgrade hardware cheaply, without rewriting firmware.

Features

  • Voice interaction — Mimo understands user intent and responds with short, clear answers. The goal is a more natural assistant experience with less friction.
  • Reaction engine — Every answer lands on four channels at once: speech, expression, LED color, and motion. Mimo doesn't just reply — it answers, feels, moves, and acts together.
  • 5-mood personality engine — Each mood changes the face expression, LED tone, motion tempo, and voice behavior. The result feels less like a control panel and more like a consistent character.
  • Camera AR — Place Mimo on your desk or in a selfie frame to blend the digital character with a live camera view. Export stickers for WhatsApp and Instagram.
  • Memory and personalization — Short-term mood, daily tasks, and long-term preferences live as separate memory layers. Mimo learns your habits over time so its character grows instead of resetting.
  • Smart home integration — Connect to lighting, temperature, and daily routines through Home Assistant. Every brand through one integration layer.
  • Safety tiers — Simple reactions run instantly, important actions ask for confirmation, and risky commands are blocked by default. API keys stay in the backend only.

Platforms

  • Live 3D web demo — Try Mimo's personality, mood system, and voice reactions in the browser right now.
  • Camera AR studio — Place Mimo on a desk, in your hand, or in a selfie frame. Export sticker assets for WhatsApp and Instagram.
  • iOS and Android apps (coming soon) — Native apps for shortcuts, routines, notifications, and continuous daily connection.
  • Physical desk robot (roadmap) — ESP32-S3 touchscreen face, 6 articulated servos, distance and touch sensors, LED ring, and voice. Upgradeable from desk base to full body.

How it’s built

  • Mimo Body — ESP32-S3 — The local controller: touchscreen face, servos, sensors, LED, and sound. Runs small commands and returns telemetry. Knows nothing about OpenAI or smart-home brands.
  • Mimo Brain — FastAPI backend — All the heavy AI: reasoning, personality, the emotion engine, command routing, memory, and the safety guard.
  • Mobile companion — React Native / PWA — Push-to-talk, mood settings, routines, smart-home shortcuts, and telemetry.
  • Desktop helper — Tauri / Electron — Project shortcuts, coding mode, reminders, and safe desktop actions.
  • Plugin / skill layer — Every feature is a modular skill (TV, AC, Home Assistant, timers, coding helper, games). A system that grows, not a hardcoded monolith.
